# Multiplication: Introduction to Arrays — Grade 2 Math

**Intellia SG | Grade 2 Mathematics**

An interactive lesson module for Grade 2 students teaching multiplication through arrays.

## Features

- 5-phase learner journey: Wonder → Story → Simulate → Play → Reflect
- 100 randomised questions across 10 game worlds
- ElevenLabs AI narration (Alice voice)
- Interactive array builder simulations
- XP, streaks, badges, and world progression
- Singapore/Western curriculum aligned

## Tech Stack

- React 18 + Vite
- CSS (no external UI library)
- ElevenLabs TTS API

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

```
VITE_ELEVENLABS_API_KEY=your_key_here
```

## Build

```bash
npm run build
```

## Deploy

Deployed on Vercel. Push to `main` branch triggers auto-deploy.
